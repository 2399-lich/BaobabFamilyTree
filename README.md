# 🌳 Baobab Family Tree

Приложение за Android, което работи изцяло офлайн, за създаване и разглеждане на вашето родословно дърво. Съхранявайте подробни профили, визуализирайте връзките между поколенията и търсете информация за цялото си семейство — всичко това се запазва локално на вашето устройство, без да е необходим интернет.

---


## Функции

- **Добавяне и редактиране на членове** — Съхранявайте име, фамилия, пол, дата на раждане, дата на смърт, място на раждане, професия, биография и снимка за всеки човек
- **Семейни връзки** — Свържете всеки член с баща, майка и съпруг/партньор
- **Визуално родословно дърво** — Плъзгане за мащабиране и плъзгане за панорамиране на платното, показващо цялото ви дърво с Безиери връзки между поколенията
- **Изглед с подробности за члена** — Докоснете всеки член, за да видите пълния му профил с кликаеми връзки към родители, съпруг/а и деца
- **Търсене** — Търсене в реално време по имена, места на раждане и професии
- **Индикатор за починали** — Членовете с дата на смъртта са визуално откроени
- **Напълно офлайн** — Всички данни се съхраняват локално с помощта на SQLite, не се изисква акаунт или интернет

## Първи стъпки

### Необходими условия

- Android Studio (препоръчва се Hedgehog или по-нова версия)
- JDK 17
- Устройство с Android или емулатор, работещ с API 16+

### Инсталиране

1. Клонирайте репозиторията:
   ```bash
   git clone https://github.com/2399-lich/BaobabFamilyTree.bit
   ```

2. Отворете Android Studio → **Файл → Отвори** → изберете папката `BaobabFamilyTree` (тази, която съдържа `build.gradle` и `settings.gradle`)

3. Кликнете **Use JVM 17**, ако ви бъде поискано

4. Изчакайте Gradle синхронизацията да приключи

5. Изберете устройство или емулатор и кликнете **Run ▶**

### Първо стартиране

При първото стартиране ще видите празно дърво. Натиснете бутона **+**, за да добавите първия член на семейството, след което го свържете с останалите, използвайки падащите менюта Father, Mother и Spouse.

## Как работи

### База данни

Всички данни се съхраняват в една SQLite таблица (`family_members`). Връзките се съхраняват като ID-та на външни ключове (`father_id`, `mother_id`, `spouse_id`) в същата таблица. Изтриването на член автоматично анулира всички препратки към него в други записи.

### Family Tree View

Дървото се изобразява в персонализиран `View`, като се използва API-то `Canvas` на Android. Членовете се разпределят по поколения чрез BFS-обход, започващ от кореновите възли (членове без известни родители). Възлите се подреждат ред по ред, а кривите на Безие свързват родителите с децата. Връзките между съпрузите са изобразени като златисти пунктирани линии. Изгледът поддържа мащабиране чрез сближаване на пръстите и панорамиране чрез плъзгане чрез `ScaleGestureDetector` и `GestureDetector`.
