# BaobabFamilyTree
Baobab Family Tree  Приложение за Android, което работи изцяло офлайн, за създаване и разглеждане на вашето родословно дърво. Съхранявайте подробни профили, визуализирайте връзките между поколенията и търсете информация за цялото си семейство — всичко това се запазва локално на вашето устройство, без да е необходим интернет.
